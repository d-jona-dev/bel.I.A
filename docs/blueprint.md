# **App Name**: Aventurier Textuel

## Core Features:

- Génération d'Aventure: Générer une narration d'aventure basée sur le monde, la situation initiale du personnage, les personnages secondaires et les actions de l'utilisateur, en utilisant un LLM comme outil.
- Chargement de modèle IA: Charger des modèles d'IA à partir de dossiers tiers.
- Interaction avec les personnages: Permettre à l'utilisateur de discuter avec des personnages fictifs ou de vivre des aventures fictionnelles.
- Chargement de Prompts: Charger des prompts contenant des informations sur le monde, la situation initiale, les personnages secondaires (nom, caractère, historique, etc.).
- Sauvegarde d'Aventure: Sauvegarde de l’aventure en format JSON, avec partitionnement des fichiers pour les personnages secondaires et les objets (armes, armures, etc.).
- Choix Multiples: Proposer des choix multiples (boutons dans l’interface) pour influencer le déroulement de l'aventure.
- Génération d'Images: Générer des portraits des personnages et des images de scènes en utilisant Stable Diffusion locale.
- Dialogue Séparé: Permettre un dialogue séparé avec chaque personnage (mode “dialogue” vs “exploration”).
- Traduction Automatique: Traduction automatique des textes générés dans différentes langues, avec un menu déroulant pour sélectionner la langue.

## Style Guidelines:

- Couleur primaire : Blanc cassé (#F8F8FF) pour un aspect propre.
- Couleur secondaire : Gris clair (#D3D3D3) pour les éléments d'interface discrets.
- Accent : Vert (#32CD32) pour les actions positives et les surlignages.
- Utilisation d'une typographie claire et lisible pour faciliter la lecture des idées.
- Icônes simples et intuitives pour représenter les différentes actions (sauvegarde, génération, etc.).
- Disposition épurée avec une hiérarchie visuelle claire pour faciliter la navigation et la compréhension.
- Animations subtiles lors de la génération des idées pour indiquer le traitement en cours.
