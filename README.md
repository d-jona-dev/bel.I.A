🚀 Getting Started
To get a local copy up and running, follow these simple steps.

---

Clone the repository:

Download ZIP

Unzip the files into a specified directory.

open a terminal.

  - cd [the path to the previously created folder]
  - npm install (Installing dependencies should only be done on first use.)
  - npm run dev


The application is not finished; please take into account the independent nature of the project.
